Создаем простую сеть
В качестве пользовательских устройств выбираем ПК\Ноутбуки
В качестве коммутаторов используем 2960
Однотипные устройств подключаем кросс-кабелем
Прямым кабелем соединяем ПК с коммутаторами
Каждому ПК назначаем IP-адрес, для примера будет использоваться 192.168.1.0/24 (маска /24=255.255.255.0)
Открываем любой ПК, переходим на вкладку Desktop и открываем IP Configuration
В отрывшемся окне вводи IP адрес в строку IP Address, маску вводим в поле Subnet Mask (автоматически подставляется 24 маска)
Командой ping проверяем сеть, для это на любом ПК необходимо перейти на вкладку Desktop и открыть Command Prompt
После чего ввести ping (например c PC2 ping 192.168.1.10)

Для удобства можно переименовать коммутаторы на схеме и в самом коммутаторе
Нажимаем на строку с именем коммутатора на схеме и меняем на любое другое (Switch0 -> SW0)
Открываем коммутатор и переходим на вкладку CLI
Нажимаем Enter
Переходим в привилегированный режим, вводим en (от слова enable)
Затем переходим в режим конфигурирования командой conf t (config terminal)
После чего вводим hostname %ИМЯ% (hostname SW0)
