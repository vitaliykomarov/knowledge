Из каждой подсети для роутера назначим первый доступный адрес, для ПК второй, третий, для коммутаторов последний возможный адрес
Каждому ПК зададим настройки в соответствии с подсетью
Настроим каждый порт GI на роутере
Настроим шлюз по умолчанию и Vlan1 (назначив IP адрес для коммутатора) на каждом коммутаторе

Проверяем сеть командой ping с любого компьютера
