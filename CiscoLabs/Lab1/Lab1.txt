Построение сети с маршрутизатором
В качестве маршрутизатора (роутер) используем 2911
Прямым кабелем соединяем ПК с коммутаторами, коммутаторы с роутером

Выделяем несколько подсетей
Каждому ПК назначаем IP-адрес и шлюз по умолчанию, например 192.168.1.1 для подсети 192.168.1.0/24
По умолчанию все ПК подключенные к коммутаторам входя в Vlan1
Настроим Vlan-ы на каждом коммутаторе с соответствующей подсетью
Переходим в привилегированный режим, вводим en (от слова enable)
Затем переходим в режим конфигурирования командой conf t (config terminal)
Набираем interface Vlan1 для перехода к конфигурации Vlan
Назначаем IP-адрес и маску подсети командой ip address 192.168.1.10 255.255.255.0
Командой exit выходим из конфигурирования Vlan
Настрои шлюз по умолчанию командой ip default-gateway 192.168.1.1

Настроим роутер, для каждого порта пропишем свою подсеть
Переходим в привилегированный режим, вводим en (от слова enable)
Затем переходим в режим конфигурирования командой conf t (config terminal)
Для конфигурирования порта GigabitEthernet0/0 вводим interface GigabitEthernet0/0
Назначаем соответствующей IP-адрес и маску ip address 192.168.1.1 255.255.255.0
Для включения порта необходимо не выходя из конфигурации GigabitEthernet0/0 ввести команду no shutdown
Так необходимо выполнить для каждого порта

Проверяем сеть командой ping с любого компьютера, например с PC0 проверим PC5
Пинг проходит
Сеть настроена
